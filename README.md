# Human Resource Management System (HRMS)

A minimal HRMS built with HTML, CSS, JavaScript, and Bootstrap.

## Features

### Authentication
- Secure login/signup
- Role-based access (Admin vs Employee)

### Employee Features
- Profile management
- Attendance tracking (check-in/check-out)
- Leave application and history
- Weekly attendance view

### Admin Features
- Leave approval workflow
- Employee management
- Dashboard with statistics
- Pending approvals overview

## Setup Instructions

1. Open `index.html` in a web browser
2. Use demo accounts or create new employee account
3. Navigate through different sections using sidebar menu

## File Structure

```
├── index.html      # Login page
├── dashboard.html  # Main dashboard
├── style.css       # Custom styles
└── script.js       # Application logic
```

## Technology Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6)
- **Framework:** Bootstrap 5.3
- **Storage:** LocalStorage (client-side)

## Key Functionalities

1. **Authentication System**
   - Login/Signup forms
   - Session management
   - Role-based navigation

2. **Attendance Management**
   - Daily check-in/check-out
   - Weekly attendance view
   - Status tracking

3. **Leave Management**
   - Leave application form
   - Leave history tracking
   - Admin approval workflow

4. **Profile Management**
   - Employee profile display
   - Department and role information

5. **Admin Dashboard**
   - Employee list management
   - Leave approval interface
   - System statistics

## Usage

1. **Login:** Use demo credentials or create new account
2. **Dashboard:** View personal stats and quick actions
3. **Attendance:** Mark daily attendance and view weekly records
4. **Leave:** Apply for leave and track application status
5. **Admin:** Approve/reject leave requests and manage employees
